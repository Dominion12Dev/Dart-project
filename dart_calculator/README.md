# Dart Calculator

A simple command line calculator built while practicing Dart basics.

## What It Does

- Adds, subtracts, multiplies, and divides numbers
- Handles invalid number input
- Prevents division by zero
- Lets the user keep calculating until they choose to exit

## Skills Practiced

- Variables and data types
- Functions
- User input with `stdin`
- Loops and conditionals
- Basic error handling
