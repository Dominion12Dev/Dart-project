# Internship Tracker

A command line app for tracking internship applications. This is a practical learning project for a student rebuilding a portfolio after changing laptops.

## What It Does

- Adds internship applications
- Shows all saved applications during the current session
- Updates the status of an application
- Deletes an application
- Uses a simple menu-based interface

## Skills Practiced

- Classes and objects
- Lists
- Functions
- Menu-driven programs
- Input validation

## How To Run

```bash
dart run bin/internship_tracker.dart
```
